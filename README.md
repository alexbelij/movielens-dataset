# Movie Lens dataset
Exploring movies rating, genre and so many more based on MovieLeans dataset.

**This is for educational purpose only.**

## Technology
- Python

## Libs
- matplotlib
- seaborn
- pandas
- numpy

## Requirements
Python 3

## Get started
If you will run it locally, run `pip install` and the libs listed in the **Libs** section.
